# Student API

## Routes
- GET / - API description
- GET /api - Swagger UI
- POST /students - Create student
- GET /students - List students
- PATCH /students - Update student by id in body
- GET /students/:id - Get student by id
- POST /auth/signup - Create user
- POST /auth/login - Login user
